Mindfulness-Based Cognitive Therapy (MBCT)

Meta-Analytic Database of Randomized Controlled Trials

Mindfulness-Based Cognitive Therapy (MBCT) is an approach that uses mindfulness meditation with cognitive behavioral therapy to treat individuals with depression and help them manage stress, anxiety, and other emotional challenges.

This online meta-analysis tool allows you to explore a simplified version of the Mindfulness-Based Cognitive Therapy (MBCT) dataset.
